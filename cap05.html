O que Não se Perde
Características do conhecimento durável

Empresas fecham, tecnologias ficam obsoletas, sectores colapsam. O que sobrevive são capacidades de pensamento, comunicação e resolução de problemas — que se transferem entre contextos. Um programador que sabe pensar em sistemas aprende uma nova linguagem em semanas. Um bom escritor adapta-se a qualquer formato.

O critério para avaliar um investimento em aprendizagem: esta competência torna-me mais valioso em múltiplos contextos possíveis? Se sim, investe. Se só serve num nicho específico de uma empresa específica, é conhecimento frágil.

As competências mais duráveis que identifiquei: pensamento crítico e análise, comunicação escrita e oral, aprendizagem rápida de coisas novas, colaboração em ambiente de incerteza, e domínio de pelo menos uma língua internacional.

Aprender Línguas
Método prático sem vender cursos

O mercado de aprendizagem de línguas está cheio de produtos que prometem fluência em 3 meses. A realidade é menos vendável: alcançar nível B2 (funcional) numa língua nova leva 600–1.200 horas de contacto efectivo, dependendo da proximidade com a língua materna. Para inglês como lusófono, é mais rápido — cerca de 600h.

O que funciona: input compreensível massivo (leitura e escuta de conteúdo que entendes 70–90%) + repetição espaçada para vocabulário + produção activa (falar, escrever). O que não funciona como método principal: gramática isolada, tradução, ou usar só apps gamificadas.

Sistema prático — do zero ao B2 em inglês
1
Fase 1 (0–100h): Vocabulário base com Anki
Descarrega o deck "English 5000" ou "Frequency Dictionary English" no Anki (gratuito). 20–30 minutos por dia de revisão de flashcards. Aprende as 2.000 palavras mais frequentes — cobrem 90% do inglês do dia-a-dia. Anki: ankiweb.net — gratuito para desktop, €24,99 para iOS (Android gratuito).
2
Fase 2 (100–400h): Input compreensível
Séries de TV com legendas em inglês (não português). Começa com séries que já conheces. Podcasts com transcrições (BBC Learning English, VOA Learning English — ambos gratuitos). YouTube com legendas activadas. O objectivo é ouvir e ler muito — não estudar formalmente.
3
Fase 3 (400h+): Produção activa
Hellotalk ou Tandem para conversar com nativos gratuitamente (troca de línguas). Italki para aulas com professores — €10–20/h. Escreve um diário em inglês. A produção acelera a fluência de forma que a leitura passiva não consegue.
💡 Para BR → inglês
Lusófonos têm vantagem: cognatos com inglês são abundantes. O maior obstáculo é psicológico — o medo de errar. A exposição consistente a inglês real (filmes, podcasts, reuniões de trabalho) resolve mais do que qualquer gramática.
Onde Estudar — Gratuito
Recursos verificados, sem afiliados
▸
Khan Academy Gratuito
Matemática, ciências, programação, finanças pessoais. Conteúdo traduzido para PT e PT-BR. Ideal para base académica ou reforço. khanacademy.org
▸
Coursera + MIT OpenCourseWare Gratuito (audit)
Cursos de universidades top (MIT, Stanford, Yale) — gratuitos em modo audit (sem certificado). Pagar o certificado raramente vale a pena. O conhecimento é o mesmo. coursera.org · ocw.mit.edu
▸
freeCodeCamp Gratuito
Programação web completa. HTML/CSS, JavaScript, Python, SQL, Machine Learning. Certificados gratuitos reconhecidos. freecodecamp.org · Canal YouTube com milhares de horas de conteúdo técnico.
▸
The Odin Project Gratuito
Desenvolvimento web full-stack, estruturado e progressivo. Um dos melhores currículos gratuitos de programação. Projecto-based. theodinproject.com
▸
Biblioteca pública / BNP / BN BR Gratuito
PT: Biblioteca Nacional de Portugal tem acesso a bases de dados e periódicos — incluindo acesso digital com cartão de leitor gratuito. BR: Biblioteca Nacional Digital (bndigital.bn.gov.br) + bibliotecas municipais com acesso a e-books.
▸
YouTube Educativo Gratuito
3Blue1Brown (matemática visual), CrashCourse (tudo), Two Minute Papers (IA), Nerdologia (ciência PT-BR), Peixe Babel (tecnologia PT-BR). Canais técnicos específicos para qualquer área profissional.
▸
Udemy Pago ~€10–15
Em promoção (frequente), cursos técnicos práticos por €9,99–14,99. Para skills específicas como Excel avançado, Power BI, Python aplicado, etc. Verifica avaliações e data de actualização antes de comprar.
Certificações que Valem
Em 2025–2026 — lista honesta

A maioria das certificações não vale o papel em que está impressa. As excepções são aquelas que o mercado usa activamente como filtro de contratação, ou que provam uma competência que é difícil de demonstrar de outra forma.

▸
Google Data Analytics Certificate ~€200 (Coursera)
Curto, prático, reconhecido. Cobre SQL, R, Tableau, análise básica. Para quem quer entrar em análise de dados sem background técnico forte.
▸
AWS / Azure / GCP Cloud Practitioner €100–300
Certificações cloud entry-level são filtros reais em muitos processos de contratação em tecnologia. AWS é o mais comum globalmente; Azure é mais predominante em Portugal.
▸
CompTIA Security+ ~€370
Padrão de entrada em cibersegurança. Muito procurado em candidaturas governamentais e grandes empresas. Vale o investimento para quem quer entrar na área.
▸
Microsoft PL-300 — Power BI Data Analyst ~€165
Para analistas de dados que usam o ecossistema Microsoft. Muito procurado em empresas portuguesas e brasileiras que usam Power BI.
Nota
Certificações não substituem portfólio. Um analista de dados sem projectos públicos e com 10 certificações é menos valioso do que alguém com 1 certificação e 5 projectos demonstráveis. Investe em ambos, mas prioriza o trabalho real.
